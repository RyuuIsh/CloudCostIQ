# 🚀 CloudCostIQ — AWS Cost Optimization Dashboard

CloudCostIQ is a Python-based AWS cost analytics and optimization dashboard that uses the AWS Cost Explorer API to track cloud spending, visualize usage patterns, and generate intelligent cost-saving insights.

## 🧠 Features

- 💰 Monthly Cost Tracking  
- 📈 Daily Cost Trend Visualization  
- 📊 Service-wise Cost Breakdown  
- 🔥 Top Expensive Service Detection  
- 📉 Forecasted Monthly Cost (with fallback logic)  
- ⚡ Optimization Recommendations (Rule-based Engine)  
- 🚨 Cost Alerts (Budget-style warnings)  
- 📊 Cost Spike Detection (Anomaly Detection)  
- 📄 Export Reports (CSV & PDF)  
- 🎨 Modern Dashboard UI (Charts + Cards + Badges)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **AWS Integration:** Boto3, AWS Cost Explorer API  
- **Data Processing:** Pandas  
- **Visualization:** Chart.js  
- **Frontend:** HTML, CSS  
- **Reporting:** ReportLab  


---

## 🧠 Optimization Engine

CloudCostIQ uses a rule-based system to detect cost inefficiencies such as:

- 🖥️ Unused EC2 Instances  
- 💾 Unattached EBS Volumes  
- 🌐 Idle Load Balancers  
- 📡 High Data Transfer Costs  
- 🗄️ Storage Optimization Opportunities  

---

## 🚨 Cost Intelligence Features

- 📉 **Forecasting** → Predicts monthly cloud cost  
- 🚨 **Alerts** → Notifies when thresholds are exceeded  
- 📊 **Spike Detection** → Identifies sudden cost increases  
- 💡 **Insights** → Converts billing data into actionable recommendations
